Detailed Description

Die ProcessList hält die Anzahl der aktuell gespeicherten process list entries und ein array zur Speicherung der tatsächlichen process list entries.

Remarks
Der Speicher für die process list entries muss vom Aufrufer bereitgestellt werden. processes verwaltet ausschließlich Referenzen auf process list entries.
