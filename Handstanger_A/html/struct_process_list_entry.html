Detailed Description

Ein ProcessListEntry besteht aus einer eindeutigen Prozessnummer (pid), einem Prozessnamen, dessen aktueller CPU Nutzung, dessen Ausführungszeit seit Start des Prozesses, die Anzahl seiner Threads, dessen Gesamtspeichergröße und dessen Status.
